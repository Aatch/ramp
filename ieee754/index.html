Low-level manipulations of IEEE754 floating-point numbers.

Installation

Add this to your Cargo.toml

[dependencies]
ieee754 = "0.2"

Examples

use ieee754::Ieee754;

// there are 840 single-precision floats between 1.0 and 1.0001
// (inclusive).
assert_eq!(1_f32.upto(1.0001).count(), 840);
